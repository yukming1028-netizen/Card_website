# 卡牌倉庫 - 快速使用指南

## ✅ 部署狀態

所有服務已成功部署並運行！

## 🎯 立即開始

### 1. 訪問網站
打開瀏覽器，訪問: **http://localhost:3001**

### 2. 搜索卡牌
- 在首頁輸入卡牌編號（例如：001）
- 按 Enter 或點擊搜索
- 查看卡牌詳細資料和統計信息

**測試用卡牌編號：**
- 001 - 火焰龍（UR, 9分）
- 002 - 冰霜鳳凰（UR, 8分）
- 004 - 暗影刺客（SSR, 8分）
- 005 - 光明牧師（SR, 6分）

### 3. 管理員登入
- 點擊右上角「管理員登入」
- **用戶名**: `admin`
- **密碼**: `admin123`

### 4. 管理卡牌
登入後可以：
- ➕ 添加新卡牌
- ✏️ 編輯現有卡牌
- 🗑️ 刪除卡牌
- 👀 查看所有卡牌列表

## ❓ 問題排查

### 無法訪問網站
1. 檢查服務狀態：`sudo docker compose ps`
2. 檢查端口 3001 是否被占用
3. 查看前端日誌：`sudo docker compose logs frontend`

### 無法登入
1. 確認用戶名：admin
2. 確認密碼：admin123
3. 查看後端日誌：`sudo docker compose logs backend`

### 搜索無結果
1. 確認卡牌編號正確
2. 檢查資料庫是否有數據
3. 查看後端 API 日誌
